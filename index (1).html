Features

Everything you need to manage
your finances

Nine core features covering authentication, transaction management, budgets, goals, charts, and reporting.

🔐
Secure Authentication
JWT-based login and registration via Supabase Auth. Passwords hashed with bcrypt. Sessions managed automatically.
Supabase Auth
💳
Transaction Tracking
Full CRUD for income and expense records. Category tagging, date filters, keyword search, and sortable table view.
Core Feature
🎯
Budget Management
Set monthly spending limits per category. Real-time progress bars. Automatic alerts at 80% and 100% thresholds.
Alert Engine
⭐
Savings Goals
Create savings targets with emoji icons, target dates, and contribution tracking. SVG ring progress indicators.
Visual Progress
📊
Dashboard & Charts
Income vs expenses bar chart, expense doughnut, budget vs actual comparison — all powered by Chart.js 4.4.
Chart.js
📄
Financial Reports
Date-range reports with KPI summary, category breakdown, and full transaction list. Export to CSV or PDF.
Export
🔔
Notifications
In-app notification centre with unread badge counter. Budget warnings and goal achievement messages.
Real-time
📱
Mobile Responsive
Fully responsive with collapsible sidebar and fixed mobile bottom navigation. Works on iPad and smartphone.
Mobile First
🔌
Offline Demo Mode
Works without a backend using in-browser sample data. Perfect for presentations without internet access.
Offline Ready
